# 🎮 Tic Tac Toe Game

## 📌 Project Overview
Tic Tac Toe is a simple 2-player game played on a 3×3 grid. Players take turns marking a cell with X or O. The player who places three of their marks in a horizontal, vertical, or diagonal row wins the game.

This project demonstrates basic frontend development concepts like DOM manipulation, event handling, and game logic implementation using JavaScript.


## ❓ Problem Statement
Beginner developers often struggle to implement interactive game logic in web applications.

This project aims to:
- Build a fully functional Tic Tac Toe game
- Implement win and draw detection logic
- Manage player turns dynamically
- Reset the game state properly


## 🎯 Features
- Two-player gameplay (Player X and Player O)
- Automatic turn switching
- Win detection (rows, columns, diagonals)
- Draw detection
- Game reset functionality
- Responsive and user-friendly UI


## 🛠️ Tech Stack
- Frontend: HTML
- Styling: CSS / Bootstrap (optional)
- Logic: Vanilla JavaScript

## 🧠 Game Logic Explanation

- The board consists of 9 cells.
- Players alternate turns.
- After every move, the game checks:
  - 3 rows
  - 3 columns
  - 2 diagonals
- If a winning combination is found → Winner is declared.
- If all cells are filled and no winner → Match is declared as Draw.
